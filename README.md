# An-lisis-y-Predicci-n-de-la-Calidad-del-Aire-en-el-GAM
Análisis y predicción de la calidad del aire en el Gran Área Metropolitana de Costa Rica, correlacionando flujo vehicular (ARESEP/CONAVI) con datos meteorológicos y contaminantes atmosféricos. Implementado con POO en Python, modelos de ML supervisado e integración de APIs públicas.
